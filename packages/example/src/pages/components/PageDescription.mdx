<PageDescription>

This is a `<PageDescription>` component. It is generally used for intro text at
the top of the page using the
[type token](https://www.carbondesignsystem.com/guidelines/typography/expressive)
`expressive-heading-03`.

</PageDescription>

## Notes

For most pages, we recommend starting with a `PageDescription` followed by
`AnchorLinks` if the content is long enough. Please note that this component,
like all MDX components, is picky about white space, the line break above and
below your content is required, and if you even have an extra space on the empty
lines above/below it wonâ€™t render correctly.
