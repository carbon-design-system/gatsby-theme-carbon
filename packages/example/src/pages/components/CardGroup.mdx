<PageDescription>

The `<CardGroup>` component is essentially a `<Row>` component with extra styling to support the `<ResourceCard>` and `<MiniCard>` components. There should be a `<Column>` inside of the `<CardGroup>`. This allows the correct border placement between a group of cards.

</PageDescription>

<InlineNotification kind="warning">

**Do not** use the `<CardGroup>` component for the `<FeatureCard>` component. In addition, while it won't hurt to use the `<CardGroup>` with the `<ArticleCard>` component, there are currently no added benefits.

</InlineNotification>

### Props

| property  | propType | required | default | description                                                                |
| --------- | -------- | -------- | ------- | -------------------------------------------------------------------------- |
| children  | node     |          |         | Needs a `<Column>` component along with a `<ResourceCard>` or `<MiniCard>` |
| className | string   |          |         | Add custom class name                                                      |
