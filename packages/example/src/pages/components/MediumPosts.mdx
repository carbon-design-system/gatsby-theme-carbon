<PageDescription>

Using a Gatsby plugin, this component automatically builds three `ArticleCards` populated with the most recent Medium posts at build time.

</PageDescription>

## Setup

To update the source account for this component, pass in a `mediumAccount` to your theme options in `gatsby-config.js`.

```js
plugins: [
  {
    resolve: 'gatsby-theme-carbon',
    options: {
      mediumAccount: 'carbondesign',
    },
  },
];
```

## Example

<MediumPosts postLimit={3} />

## Code

```jsx path=components/MediumPosts/MediumPosts.js src=https://github.com/carbon-design-system/gatsby-theme-carbon/tree/master/packages/gatsby-theme-carbon/src/components/MediumPosts
<MediumPosts postLimit={3} />
```

## Props

| property  | propType | required | default             | description                                                                              |
| --------- | -------- | -------- | ------------------- | ---------------------------------------------------------------------------------------- |
| cardProps | object   |          | `{{color: 'dark'}}` | These props are passed through to the internal [`ArticleCard`](/components/ArticleCard). |
| postLimit | number   |          | `3`                 | Sets the number of posts pulled from Medium, default is `3`, maximum is `10`.            |
